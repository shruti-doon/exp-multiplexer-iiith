Building a 2 x 1 MUX
Simulation of a 2 x 1 MUX built using logic gates.

Start- Starts the simulation after the inputs are properly selected. Default speed is 1x Select the speed through dropdown menu

Reset - Resets the simulation

Click on the green input bits to set their values.


Input1(I1) is bit 1, Input2(I2) is bit 2 and Select(S) is select line
Output(O)
Logic of multiplexer is: O = (S')*(I2)+(S)*(I1)
Eg: I1 is 1 , I2 is 1 and S is 0. Then the output(O) is 1
If S=1, O=I1
If S=0, O=I2
